# 开发准备工作

## 前后端分离

- 服务端
    - 数据处理
    - 高性能高并发
- 前端
    - 用户体验，流畅性
    - 操作性

## 服务端

- SpringBoot or Spring Cloud (微服务)
    - Spring Cloud or nginx or dubbo
- Redis
- fastdfs or other, 分布式存储
- elk, 搜索
- logstash, 日志
- RabbitMQ
- 工作流
- 自动任务

## 开发规范

- java 开发规范
    - 参考阿里java开发手册，形成公司开发规范
    - 针对开发工具，有相应的插件进行规范检查
- js 开发规范
    - 已有标准规范参考，并且有工具支持
- 其他语言
    - 待定
